### Prerequisites
- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio or VS Code
- Git

### Installation

1. Clone the repository:
```bash
https://github.com/AbdoJoker99/xo.git
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the application:
```bash
flutter run
```
